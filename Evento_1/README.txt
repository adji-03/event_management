*************** APP IMPORTANT NOTES ********************* 
	|


1. Gestion des Événements:

   - addEvent.form, addEvent.java, addEventButton.java : Fichiers pour ajouter un événement.

   - EditEvent.form, EditEvent.java : Fichiers pour modifier un événement.

   - Event.java, eventDescriptionText.java : Fichiers pour la gestion des données d'événement.

   - eventpagetuser.form, eventpagetuser.java : Pages pour les utilisateurs concernant les événements.



2. Administration :

   - adminpage.form, adminpage.java : Pages pour l'administration.

   - gestioninscrit.form, gestioninscrit.java : Gestion des inscrits aux événements.



3. Connexion et Interface Utilisateur :

   - login.form, login.java : Pages de connexion.

   - Home.form, Home.java : Page d'accueil.




4. Base de Données :

   - DatabaseConnection.java : Connexion à la base de données.

5. Authentification :

   - login.form, login.java, loginpage1.jpg : Fichiers pour la page de connexion.

   - signup1.form, signup1.java : Fichiers pour l'inscription des utilisateurs.



6. Gestion des Utilisateurs :

   - User.java : Fichier pour la gestion des données utilisateur.



7. Autres:

   - Main.java : Fichier principal du projet.

   - progress.form, progress.java : Fichiers pour le chargement de la page.

   - resultSet.java : Fichier pour la gestion des résultats de la base de données.
